Chocolatey Install Script

It should be possible to drop this into a bat script and run it in a command window with Administrative privileges. It will retrieve some basic tools a .Net developer might want.

You can omit the first line if you already have Chocolatey installed.

@powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" && SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin choco install git choco install 7zip choco install astrogrep choco install rdcman choco install gow choco install winmerge choco install keepass choco install qttabbar choco install cutepdf choco install foxitreader choco install sublimetext2 choco install fiddler choco install hxd choco install spacesniffer
